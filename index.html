Парсер сайта сети ресторанов Farfor
  • Стэк: Python, requests, BeautifulSoup, openpyxl
  • Описание: Парсер для отслеживания скидок и цен на торговые позиции. Сперва получает список городов, предлагает пользователю выбрать свой. Затем идет на поддомен и парсит все категории. В конце запрашивает формат вывода. Это может быть или JSON или XLSX. При сохранении в XLSX создает новый лист с датой обращения, выгружает туда все данные и окрашивает их диапазоном красный-зеленый в зависимости соотношения цена/вес и цена/кол-во для сетов. При отсутствии файла и при первом использовании создает в рабочем каталоге файл формата "ГОРОД".xlsx. Может быть полезно для мониторинга цен в других городах. Упакован в .exe.
  • Код: Тык
Телеграм-бот для расчета монтажа электропроводки
  • Стэк: Python, aiogram, SQLite3, SQLAlchemy
  • Описание: Подобие коммерческого проекта. Во время работы инженером в энергетической компании поступило предложение написать бота для отсеивания лидов и разгрузки мастера. По требованию заказчика пользователь вносит свои примерные расчеты через кнопочный интерфейс, эти данные считаются по прайсу, который заказчик со своего аккаунта подгружает в бота. Далее пользователь получает примерный расчет работ и предложение отправить этот расчет мастеру с автоматическим оформлением заявки. Мастер получает полную информацию о заказе и объекте и связывается с заказчиком в телеграм.
